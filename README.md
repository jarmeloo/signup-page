## 📝 Signup Page – Formulário de Cadastro

Uma página de cadastro moderna e elegante, desenvolvida com **HTML e CSS**, focada em uma experiência visual limpa e agradável para o usuário, utilizando efeitos gradientes suaves e animações sutis.

---

## ✨ Visão Geral

A **Signup Page** apresenta um formulário de criação de conta com campos essenciais, como nome completo, e-mail e senha. O layout foi pensado para ser simples, intuitivo e visualmente atraente, sem o uso de JavaScript ou bibliotecas externas.  
Todo o design é construído apenas com CSS, explorando gradientes, sombras, blur e transições para criar uma interface moderna.

---

## 🎯 Funcionalidades

* 🧾 Formulário de cadastro com campos obrigatórios
* 🌈 Fundo em gradiente suave
* 📱 Layout centralizado e responsivo

* **CSS**

## 📝 Observações

Este projeto foi desenvolvido para aprendizado, experimentação visual e prática de animações em CSS, podendo ser adaptado para outros projetos. Não realiza autenticação real de usuários. Não possui backend ou persistência de dados. Pode ser utilizado como base visual para projetos maiores
